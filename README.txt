COMMENT EXECUTER LE PROJET ?

Etape 1 : S'assurer que la dernière version de node.js est installée sur votre machine.
Etape 2 : Ouvrir le fichier "index.js" dans le dossier "pages" et remplacer l'adresse ip à la ligne 13 par votre adresse IPv4
          de votre réseau.
Etape 3 : A partir d'un terminal, se rendre à la racine du projet et exécuter la commande "npm install", 
          garder le terminal ouvert.
Etape 4 : Une fois toutes les dépendances installées, Exécuter le fichier "serveur.py" qui se trouve à la racine du projet.
Etape 5 : A partir du terminal précédement ouvert, exécuter la commande "npm run dev".

Il vous suffira de d'ouvrir une nouvelle page dans le navigateur de votre choix et de vous rendre sur 
l'une des pages suivantes :
    - http://localhost:3000/
    - votreAdresseIPv4:3000/
